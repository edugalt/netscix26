# Meso-scale organization of networks: theory and applications

This repository includes the material to be presented at the [International School and Conference on Network Science NetSciX 2026](https://netscix2026.github.io/) in Auckland, New Zealand, during the period 17th-20th of February 2026. It will be part of the course "Meso-scale organization of networks: theory and applications" by [Eduardo G. Altman](https://www.maths.usyd.edu.au/u/ega/).

In addition to the notebooks contained in this repository, additional coding resources to be discussed are:
- [The NetworkX Python library](https://networkx.org/en/).
- [The Graph-tool Python library](https://graph-tool.skewed.de/)
- Results from the repository on detection [Community Types](https://github.com/communityType)
- [TopSBM: Topic Models based on Stochastic Block Models](https://topsbm.github.io/)
